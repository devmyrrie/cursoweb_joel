* No especifica el tipo de documento, ni el encabezado HTML.
Si no se usa la especificación del documento, el navegador decide usar el modo compatibilidad y puede ser que decida ignorar algunas especificaciones por ende ignora partes del documento.

* Algunas etiquetas <p> están cerradas y otras no.
La etiqueta p  no puede tener anidado ningun elemento tipo block. Si no se cierra el considera que se esta anidando algo, e inmeditamente cierra el tag p. Por lo que los proximos tags no van a ser mostrados como deben ser.

Recordar que dentro de inline no puede tener anidados blocks.
